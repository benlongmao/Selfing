# Release notes

## v0.1.0 (tag)

**Audience:** Researchers and builders comfortable with **experimental** software who want a **runnable** setup to explore **functional selfhood** in a **single long-lived** LLM instance (memory, `z_self`, layered rules, rhythm).

**What to expect**

- English-first UI and docs; Apache-2.0.
- Install path: `install_s_project.sh` → configure `.env` → `manage_services.sh start` → http://localhost:8080

**Known rough edges**

- Some subsystems are strong; others are provisional or historical layers.
- High-risk tools (shell, optional evolution) require careful configuration; see `docs/security-notes.md`.
