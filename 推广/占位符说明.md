# 占位符 — 发布前全局替换

在以下所有 Markdown 里搜索并替换（可用编辑器全局替换）：

| 占位符 | 换成 |
|--------|------|
| `YOUR_REPO_URL` | 例如 `https://github.com/yourname/s-main` |
| `YOUR_REPO_PATH` | 例如 `yourname/s-main` |
| `YOUR_HANDLE` | 你的 Twitter/X / GitHub @id |
| `YOUR_EMAIL` | 公开联系邮箱（可选） |
| `YOUR_DEMO_URL` | 若有公开 demo 或录屏链接；没有则删掉相关句或换成「录屏待补」 |
| `YOUR_ZH_REPO_URL` | 中文主仓地址（若有） |

**建议**：在 GitHub 仓库 **About** 里填 Website（可以是录屏 YouTube / 飞书 / 任意稳定链接），发帖时统一用同一 URL。
