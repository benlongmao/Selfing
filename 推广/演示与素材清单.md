# 演示与素材 — 你要准备的文件清单

把成品放在仓库内或任意稳定 URL；**推广帖只贴一个主链接**（建议 GitHub + 录屏二选一为「主入口」）。

## 建议最小集

| 素材 | 规格 | 用途 |
|------|------|------|
| UI 截图 1 张 | PNG，宽 1200～1600 | README、HN、知乎 |
| 「中央循环」图 | 可用 README 现有 ASCII/图导出 | Twitter、博客 |
| 60～120 秒录屏 | 无声音也可；展示 install + 打开页面 | HN、Reddit 首评 |

## 录屏脚本（按顺序）

1. 终端：`git clone` + `cd` + 安装脚本（或你文档里的最短路径）  
2. 启动服务 + 浏览器打开 `localhost:8080`  
3. 发一条简单对话或展示设置页（10 秒）  
4. 结尾画面：停 3 秒显示 **GitHub URL 文字叠加**（防链丢）

## 可选进阶

- 15 分钟 walkthrough：架构一页 + `chat_service` 主路径（面向开发者）
- `docs/promo/` 目录：集中放图，README 引用相对路径

## 本目录

可在 `s-main/推广/assets/` 下自行建文件夹放图（勿提交密钥）；若图片很大，考虑 **Git LFS** 或外链图床（稳定性优先）。
