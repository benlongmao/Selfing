# 发帖前 / 发帖后检查清单

## 发帖前（至少打勾 15 条再发 Show HN / Reddit）

- [ ] `YOUR_REPO_URL` 在浏览器无痕模式打开，首页无 404
- [ ] README 首屏可见：**是什么、谁该用、怎么跑、什么不是**
- [ ] Quick Start 命令你本地 **从 clone 跑通一遍**（或写明最低环境）
- [ ] License 文件存在且与 About 一致（Apache-2.0）
- [ ] `.env.example` 存在（无密钥）
- [ ] Topics 已填
- [ ] 有一张 **静态图** 在 README（或 Release 说明里）
- [ ] 有 **Release / tag**（哪怕 0.1）
- [ ] `SECURITY.md` 或 issue 里至少一种安全联系路径（可选但推荐）
- [ ] 诚实局限段已复制到剪贴板（备评论用）
- [ ] 与 LangGraph /「角色扮演」对比 **一句** 已背熟
- [ ] 发帖后 **2 小时** 你有空回评论（写进日历）
- [ ] 中文仓与英文仓 **互链**（若双仓公开）
- [ ] 本推广目录占位符已替换（避免发出去还带 `YOUR_REPO_URL` 字面量）
- [ ] 录屏或截图链接 **HTTPS** 可公网访问

## 发帖后 24 小时内

- [ ] 每条 **严肃技术质疑** 都回（哪怕简短「已记录 issue#」）
- [ ] 若发现 README 致命 typo：**小步提交**修正并可在评论贴「fixed link typo」
- [ ] star / fork 数截图存证（可选，自用）

## 发帖后一周内

- [ ] 把 FAQ 里重复问题 **写进 README 或 docs**
- [ ] 若某平台效果好：记录来源，下次优先深耕

## 负面情况

- [ ] 若被 flag / 删帖：**不**立刻换号重发；复盘标题是否夸大、账号是否过新
- [ ] 若出现人身攻击：**不**纠缠；只回应事实与链接
