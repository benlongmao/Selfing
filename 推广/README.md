# Selfing（s-main）推广资料包

本目录包含**可照抄执行的文案、分平台说明、周计划与检查清单**。按顺序做即可。

## 使用前必读

1. 打开 [`占位符说明.md`](占位符说明.md)，把全文里的 `YOUR_REPO_URL`、`YOUR_HANDLE` 等换成你的真实信息。
2. 按 [`计划-第1-4周.md`](计划-第1-4周.md) 执行；每周不必全做，但**顺序不要颠倒**（先 GitHub 门面，再外链发帖）。

## 与中文主仓的关系

对外说明建议统一为：

- **英文 / 国际发现 / PR 引用**：以 `s-main` 为主入口。
- **中文讨论与长文档社区**：保留原 `s` 仓，并在两仓 README 顶部互相链接（具体句子见 `GitHub-具体操作.md` 末节）。
