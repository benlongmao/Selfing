## 英文 elevator pitch（2～3 句，推文 / 邮件首段）

```
Selfing (s-main) is an experimental Python runtime that connects one LLM instance to persistent memory, internal state (z_self), layered self rules (L0/L1/L2), reflection, and background rhythm. The design question is practical: what engineering loops let “what happened before” shape the next answer—so the system behaves like one continuing subject rather than a disposable chat call. It explicitly does not claim phenomenal consciousness; it makes functional continuity and self-reference testable in code.
```
