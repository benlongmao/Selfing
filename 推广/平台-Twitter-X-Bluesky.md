# Twitter / X / Bluesky — 英文发帖指南

## Twitter/X

### 单条（≤280 字符）— 适合转发仓库 Release

```
Selfing (s-main): Apache-2.0 runtime for *one* long-lived LLM instance—memory, z_self state, L0/L1/L2 rules, reflection, rhythm. Continuity over disposable calls. Experimental. YOUR_REPO_URL
```

### Thread（3～5 推）

1. **Hook**  
   `Most AI stacks optimize “finish the task.” We’re experimenting with “stay one subject across time.”`

2. **What it is**  
   `Selfing / s-main: Python runtime—SQLite memory, z_self vector, layered rules (incl. reflection→L2), other-model, intent markers, background ticks.`

3. **Link**  
   `Repo + quick start: YOUR_REPO_URL`

4. **Scope**  
   `Explicitly not claiming phenomenal consciousness. Functional continuity + ethics for traces.`

5. **Ask**  
   `If you run local LLMs: what continuity hooks do you wish existed?`

### 标签

`#LLM` `#opensource` `#Python` `#AI` — 少用一堆标签，像机器人。

### 互动

- 引 **1 张图**（循环图或 UI）可明显提高传播。
- 若有人骂「玄学」：用 HN 预案里那段 **Scope of Claims** 回复，不拉长战。
